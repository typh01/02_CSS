화면 배치(표시, 형식) 지정 속성 : display

    요소가 화면에 어떻게 표시될지 형식을 지정하는 속성

    
    - block : 화면/영역을 행으로 분할(수직 분할)
      1) 한 행을 요소 하나가 전부 차지함
        -> 다음 요소는 다음 줄로 넘어감

      2) 크기(width, height) 지정 O


    
    - inline : 화면/영역을 열로 분할(수평 분할)
      1) 한 행을 내용 크기 만큼 열로 분할

      2) 크기(width, height) 지정 X


    
    - inline-block 
      : 화면/영역을 열로 분할(inline) + 크기 지정 O(block)
    
    - none : 화면에 표시 X
      -> 화면에 안보일 뿐이지 실제로는 존재함!

    - flex : 요소가 정렬되는 방향
            요소간의 간격을 유연하게 처리하여 표시
            (브라우저가 간격, 크기를 계산해서 실시간으로 적용)
        + (1차원 직선형태)
    
    - grid : 격자 무늬 형식으로 화면을 쉽게 분할
        + (2차원 평면 형태)
