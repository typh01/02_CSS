CSS 선택자 우선 순위

    - CSS 는 기본적으로 style 태그 또는 .css 파일에
      작성된 순서대로 해석이 진행되지만
      (왼쪽 -> 오른쪽, 위 -> 아래)

      같은 요소에 여러 css 속성이 설정되는 경우
      위치, 선택자에 따라서 우선 순위가 결정된다.

    
      [꼭 알고 있어야 하는 내용]
    

    1) 동일한 우선 순위로 css 속성이 설정된 경우
      -> 제일 "마지막"에 작성된 CSS 속성이 반영된다!
        (마지막 내용으로 덮어쓰기 된다!)

    2) 여러 선택자를 이용해서 하나의 요소에 
       여러css 속성을 적용해도
       css 속성이 다르면 모두 반영된다.
      (곂치는게 없으니까 덮어쓰기 X)
      
    

    (1순위 최우선!!)
    1순위 : !important
    
    2순위 : inline-style (요소에 style 속성을 작성)
    
    3순위 : 아이디 선택자 (#아이디 속성값)
    
    4순위 : 클래스 선택자 (.클래스명)
    
    5순위 : 태그 선택자 (태그명)
