요소의 영역(여백) 관련 스타일

    HTML 요소는 총 4가지의 영역으로 구성되어 있음

    1.content(내용)
      - 요소의 내용이 작성되어지는 영역
      (시작 태그와 종료 태그 사이에 작성되는 
       내용, 값, 이미지를 생각하면 됨)

    2. padding
      - content와 border 사이 공간

    3. border(테두리)
      - 요소의 테두리가 지정되는 영역
      - content보다 바깥쪽에서 감싸고 있는 형태


    4. margin(여백, 간격)
      = 다른 요소와의 간격을 나타내는 영역

      

      
box-sizing


      - width/height 크기 설정 시
        크기가 적용될 범위를 지정하는 속성

        1. content-box (기본값)
          - width/height 설정 시
            content 영역의 크기만 설정

        2. border-box
          - width/height 설정 시
            설정된 값이 content + padding + border 크기를'
            합산한 결과가 되도록
            content 영역의 크기를 자동으로 조정

padding, border, margin 원하는 방향만 설정하기 

    000-top    : 윗쪽 크기
    000-bottom : 아랫쪽 크기
    000-left   : 왼쪽 크기
    000-right  : 오른쪽 크기

    단, border는 [크기 스타일 색상] 3가지를 순서대로 입력
방향 테스트

padding, margin, border-width 한 번에 여러 방향 지정하기

    padding 또는 margin 또는 border-width : 크기;

    크기 1개 작성 : 상하좌우    적용
    크기 2개 작성 : 상하 / 좌우 적용
    크기 3개 작성 : 상 / 좌우 / 하 적용
    크기 4개 작성 : 상 / 우 / 하 / 좌 적용(시계방향)
방향 테스트

margin : auto;

    다른 요소와의 간격을 자동으로 설정하여
    가운데 정렬하게 만드는 속성
